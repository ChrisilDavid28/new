Andhra Pradesh

A.P. is a state situated on the southeastern coast of India. It is India's fourth largest state by area and fifth largest by population.

  • Districts: 23
  • Capital City: Hyderabad
  • Largest City: Hyderabad
  • Governor: Biswabhusan Harichandan
  • Chief Minister: Y. S. Jaganmohan Reddy
  • Population: 78,323,330
  • Tourist spots: Tirumala Tirupati, Charminar, Golconda Fort, Chandragiri, Chowmahalla Palace, Falaknuma Palace, etc.
